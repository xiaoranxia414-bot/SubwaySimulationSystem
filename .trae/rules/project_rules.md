# 项目 AI 规则

本项目已加载 superpowers-zh (AI 编程超能力)。在处理任务时，请严格遵守以下工作流：

1. **复杂任务先查阅 skills**：处理非琐碎任务前，请先读取 `.trae/rules/` 目录下对应的 skill 说明（如 `writing-plans.md`, `systematic-debugging.md`）。
2. **执行顺序**：
   - 优先进行 `brainstorming` (头脑风暴) 明确需求。
   - 使用 `writing-plans` 编写详细实施计划。
   - 执行每一步后，参考 `verification-before-completion` 进行验证。
3. **语言要求**：所有输出（计划、解释、注释）请优先使用中文。
